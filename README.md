# DigitalPersona-4500-Angular
This is the Angular implementation for the DigitalPersona 4500 fingerprint reader. It does not have styles, as the sole purpose is to showcase the project's logic. You can observe methods and configurations from the @digitalpersona/devices library. It was taken as a reference from: https://github.com/hidglobal/digitalpersona-sample-angularjs/tree/master/src
